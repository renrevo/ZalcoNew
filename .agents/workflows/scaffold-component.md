# Milestone-Based Component Scaffold

## Steps:
1. **Milestone 1 - Structure:** Generate the semantic HTML/JSX skeleton of the component based on my prompt. Pause and wait for my approval.
2. **Milestone 2 - Styling:** Apply bold, minimalistic styling following our workspace design rules. Pause and wait for my approval.
3. **Milestone 3 - Logic & State:** Implement interactive states, animations, or data-binding required for the component. 
4. **Milestone 4 - Edge Cases:** Add error states, loading skeletons, and ensure mobile responsiveness.

**Negative Prompts to Enforce:** - Exclude unnecessary wrapper `div`s.
- Exclude generic CSS box-shadows or gradients; use custom, refined values.
